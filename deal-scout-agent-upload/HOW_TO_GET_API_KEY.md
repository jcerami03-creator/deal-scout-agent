# How to Get Your SerpAPI Key

1. Go to https://serpapi.com/
2. Sign up or log in.
3. Open your SerpAPI dashboard.
4. Copy your private API key.
5. Open the `.env` file in this project.
6. Replace `paste-your-serpapi-key-here` with your real key.

The final `.env` file should look like this:

```text
SERPAPI_API_KEY=your-real-key-here
```

Do not put your SerpAPI password in this file. Only use the API key.
